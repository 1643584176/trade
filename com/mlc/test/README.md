# AI量化交易系统

这是一个基于Python和MetaTrader 5的AI量化交易系统，具备以下特点：

## 功能特性

1. **M15数据用于1小时交易** - 系统使用M15周期数据，但每小时执行一次交易决策
2. **自主进化能力** - 模型具备自我进化机制，根据表现自动调整
3. **奖惩机制** - 根据模型表现进行奖励或惩罚
4. **时间特征和K线新特征** - 包含丰富的时间和K线特征
5. **完全自动化** - 支持无人值守的自动交易

## 文件结构

- `mt5_connector.py` - MT5连接器
- `feature_engineer.py` - 特征工程模块
- `ai_model.py` - AI模型（具备自主进化能力）
- `backtester.py` - 回测模块
- `train_model.py` - 模型训练主程序
- `real_time_trader.py` - 实时交易模块
- `config.py` - 配置文件

## 安装依赖

```bash
pip install MetaTrader5 pandas numpy scikit-learn
```

## 使用方法

### 1. 模型训练

```bash
python train_model.py
```

这将：
- 生成2025年的示例数据（实际使用时会从MT5获取）
- 进行特征工程
- 训练AI模型
- 运行回测
- 保存训练好的模型

### 2. 实时交易

```bash
python real_time_trader.py
```

这将：
- 加载训练好的模型
- 连接到MT5
- 实时获取数据并做出交易决策
- 执行交易

## 配置说明

在 `config.py` 中可以修改各种参数：
- MT5连接参数
- 交易参数（品种、手数、止损等）
- 模型参数
- 回测参数

## 注意事项

1. 实际使用前需要正确配置MT5连接参数
2. 确保MT5终端已经启动并登录
3. 根据实际交易品种调整参数
4. 建议先使用模拟账户测试

## 自主进化机制

模型采用代数概念，根据每代的表现决定是否进化：
- 表现提升：进入下一代
- 表现下降：保持当前代数
- 性能历史会被记录用于分析

## 风险提示

交易有风险，使用前请充分测试！
